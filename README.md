
# Flutter Image Gallery

A simple android image gallery viewer.



## Deployment

To deploy this project run

```bash
  flutter build apk --release
```


If you have connected android device, run

```bash
  flutter install --release
```



## Features

- Folder Separation
- Full Screen Image Viewer
